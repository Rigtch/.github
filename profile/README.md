# Welcome to Rigtch!

## What is rigtch.fm?

rigtch.fm is a web application focused on displaying your all-time statistics such as favorite artists, most listened songs etc.

## How to use it

Explore your statistics on https://rigtch-fm.vercel.app.

## Repositories

- [rigtch-fm-www](https://github.com/Rigtch/rigtch-fm-www)
- [rigtch-fm-api](https://github.com/Rigtch/rigtch-fm-api)
- [rigtch-bot](https://github.com/Rigtch/rigtch-bot)
