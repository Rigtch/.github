# Welcome to Rigtch

Real time spotify statistics calculation based on your listening history.

## How to use it

Explore your statistics on <https://rigtch-fm.vercel.app>.

## Repositories

- [rigtch-fm-www](https://github.com/Rigtch/rigtch-fm-www)
